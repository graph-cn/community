<h1 align=center>图社区</h1>

## 关于图社区
图社区是一个对图相关信息技术的学习、交流、分享的组织。

包括但不限于图数据库、图算法、图可视化等。

初衷在于协助完善现有图技术相关的生态，并探索图技术的应用场景，包括可落地的载体、业务领域等。
